# Документация к API "Работа для всех"

<img src="pictures\1732094330925.png" alt="На сером фоне надпись в три строки: Работа для всех, API и Работа для всех по брайлю" width="500" height="500">

API "Работа для всех" — это открытый REST API для поиска вакансий, подходящих для людей с инвалидностью в России. Проект находится в активной разработке.

API построен на базе открытых API портала [Работа России](https://trudvsem.ru/) и сайта поиска вакансий [hh.ru](https://hh.ru/).

Познакомиться с работой API можно в Telegram-боте: [Работа для всех](https://t.me/work_for_all_people_bot).

## Основные возможности

- Поиск вакансий по населенному пункту
- Поиск по ключевым словам среди найденных вакансий
- Просмотр информации о вакансиях
- Получение детальной информации о конкретной вакансии
- Получение списка всех регионов и фильтрация по федеральным округам
- Получение списка федеральных округов

## Архитектура

- **Технологии**: FastAPI, PostgreSQL, SQLAlchemy, Docker
- **Основные модули API**:
  - `/api/v1/vacancies/` — работа с вакансиями (поиск, список, детали)
  - `/api/v1/regions/` — работа с регионами
  - `/api/v1/federal-districts/` — работа с федеральными округами
- **Интеграция с внешними API**: загрузка вакансий с hh.ru и trudvsem.ru

## Запуск проекта

1. Клонируйте репозиторий и установите зависимости (`pip install -r requirements.txt`).
2. Запустите проект через Docker Compose: `docker-compose up --build`
3. При первом запуске автоматически загружаются и сохраняются данные о регионах РФ. Без успешной загрузки данных о регионах работа API невозможна.
4. После успешного старта API будет доступен на порту 8000.

**Важно:** проект находится в стадии разработки, структура и функционал могут меняться.
